MongoDB Update Operators

$set updates the value of a field. If the field does not exist, it creates it.
Query
$unset removes a field from the document entirely.
Query
$inc increments (or decrements with a negative value) a numeric field by the given amount.
Query
$rename renames a field. The old field is removed and a new field with the new name holds the same value.
Query
$push appends a value to an array field. If the field does not exist, it creates a new array.
Query
$pull removes all occurrences of a value from an array field.
Query
updateOne updates only the first matching document. updateMany updates ALL matching documents at once.
Query
