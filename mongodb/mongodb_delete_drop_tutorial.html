MongoDB gives you 3 main ways to remove data:
deleteOne() — remove exactly 1 matching document
deleteMany() — remove all matching documents
drop() — delete the entire collection

deleteOne()

Removes the first document that matches the filter. Only one document is deleted even if many match.

Example 1 — delete by name
db.students.deleteOne({ name: "Sara" }) // Finds Sara and removes her document
Result will appear here…
Example 2 — delete by grade
db.students.deleteOne({ grade: "C" }) // Deletes only ONE student with grade C (Emma)
Result will appear here…

Even if multiple documents match, deleteOne() stops after the first.

deleteMany()

Removes all documents that match the filter. If you pass {} as the filter, it clears the entire collection (but keeps the collection itself).

Example 1 — delete all CS students
db.students.deleteMany({ course: "CS" }) // John and David both get deleted
Result will appear here…
Example 2 — delete all students from Delhi
db.students.deleteMany({ city: "Delhi" }) // Ali and John are from Delhi → both deleted
Result will appear here…
Example 3 — delete all documents
db.students.deleteMany({}) // Empty filter → deletes every document // Collection still exists, just empty
Result will appear here…

deleteMany({}) empties the collection but does NOT remove it from the database.

drop()

Removes the entire collection — all documents AND the collection itself. This cannot be undone.

⚠ drop() is irreversible. The collection, its documents, and all its indexes are permanently gone.
db.students.drop() // Returns true if collection existed and was dropped // Returns false if collection did not exist
Result will appear here…

Verify it's gone:

db.getCollectionNames() // "students" will no longer appear in the list

Quick reference — when to use each operation:

Operation What it removes Collection survives? Indexes survive?
deleteOne(filter) First matching document Yes Yes
deleteMany(filter) All matching documents Yes Yes
deleteMany({}) All documents Yes (empty) Yes
drop() Entire collection No No

Return values:
deleteOne / deleteMany → { acknowledged: true, deletedCount: N }
drop() → true or false

Use case guide:
Remove one specific record (e.g. a student who left) → deleteOne
Remove a group (e.g. all failed students) → deleteMany
Wipe all data but keep the collection → deleteMany({})
Completely remove the collection → drop()
