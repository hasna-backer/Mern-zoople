MongoDB Query Operators

Comparison operators filter documents by comparing field values. Use $eq, $ne, $gt, $gte, $lt, $lte.
Query
Logical operators combine multiple conditions. $and requires all conditions, $or requires at least one, $not negates, $nor matches when none are true.
Query
$exists checks if a field is present. $type filters by BSON type (e.g. "string", "int", "double").
Query
$in matches any value in an array. $nin matches values NOT in the array.
Query
$regex filters documents where a string field matches a regular expression. Optionally use $options: "i" for case-insensitive matching.
