Exercises
  1. Set width to 200, padding 20, border 10. What is total width with content-box?
  2. Toggle to border-box. How does the total width change?
  3. Set margin to 40 and observe spacing from nearby elements.
Answers:
  • content-box: total width = content width + 2×padding + 2×border = 200 + 40 + 20 = 260px.
  • border-box: total width = 200px — padding and border are included inside the 200px.
  • Margin does not affect the element's box but adds space outside it (separates from neighbours).

Lesson notes (for instructors)

  • Start by showing the red "Content" area and explain the content box.
  • Increase padding: ask students what happens to visual space inside the border.
  • Change border width/style and show how it sits around padding/content.
  • Switch box-sizing and explain the difference in how width is interpreted.
  • Give the exercises in the control column and ask students to compute expected total widths before verifying with the demo.
